## Angular Material  + Webpack + Node Express sample


### How to run?


```bash

# Install dependencies and global gulp
$ npm install && npm install -g gulp

# Webpack plugin to pack everything
$ gulp pack

# Run nodemon server and open browser
$ gulp

```
